# フレミングの法則シミュレーション

## 概要

このプロジェクトは、**フレミングの左手の法則**を視覚的にシミュレーションするインタラクティブなWebアプリケーションです。磁場と電流の相互作用により生じる力の方向を、リアルタイムで図解表示します。

## フレミングの法則とは

フレミングの法則は物理学において、磁場の中を流れる電流に作用する力の関係を表します。

### 左手の法則
左手の親指・人差し指・中指を互いに直角になるように立てたとき：
- **人差し指** = 磁場の方向（B）
- **中指** = 電流の方向（I）
- **親指** = 力の方向（F）

### 数式
$$\vec{F} = \vec{I} \times \vec{B}$$

力の大きさ：
$$|F| = |I| \times |B| \times \sin(\theta)$$

## 機能

### 1. インタラクティブな図解
- リアルタイムで磁場、電流、力の矢印を表示
- 3次元空間での方向関係を2次元に投影して表現
- 色分け表示（磁場：赤、電流：青緑、力：緑）

### 2. パラメータ調整
- **磁場方向**：6方向から選択（右、左、上、下、手前、奥）
- **電流方向**：6方向から選択
- **磁場の強さ**：0.5～3.0の範囲でスライダー調整
- **電流の強さ**：0.5～3.0の範囲でスライダー調整

### 3. ベクトル情報表示
- リアルタイムで各ベクトルの3成分を数値表示
- 外積計算による力のベクトルを自動計算

## 使い方

### 基本操作
1. **方向を選択**：上下のボタングループで磁場と電流の方向を設定
2. **強さを調整**：スライダーで各要素の大きさを調整
3. **結果確認**：キャンバスにリアルタイムで結果が表示されます

### 実験例
- 磁場を「右」、電流を「上」に設定 → 力は「手前」方向
- パラメータを変更 → 力の方向が即座に更新される

## 技術仕様

### Vector3クラス
3次元ベクトルを表現し、以下の機能を提供：
- 外積計算（`Vector3.cross()`）
- 大きさの計算
- 正規化
- スケーリング

### 座標系
- **3D座標**：x（横）, y（縦）, z（奥行き）
- **2D表示**：Canvas上でx-y平面、z軸は斜め表示

### 色彩スキーム
- 磁場（B）：`#ff6b6b`（赤）
- 電流（I）：`#4ecdc4`（青緑）
- 力（F）：`#00ff88`（緑）

## 物理学的背景

### 磁場と電流の相互作用
- 磁場 B の中を電流 I が流れると、その導体に力 F が生じます
- この力の方向はフレミングの法則で決定されます
- 力の大きさは電流の強さと磁場の強さに比例します

### 実際の応用
- **電動機**：電磁力を利用した動作原理
- **スピーカー**：磁場と電流から音声を生成
- **クレーン**：電磁石による吊上げメカニズム
